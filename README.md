# TaskOrbit - Project Management Frontend

The frontend client for TaskOrbit, a comprehensive task and project management application designed to help teams track assignments, collaborate on projects, and analyze team performance. Built with React and Vite.

---

## Quick Start

Follow these steps to run the client locally:

```bash
git clone https://github.com/ShubhamManeSM/taskorbit
cd taskorbit
npm install
npm run dev
```

## Technologies
- **React** & **Vite** (Core framework & bundler)
- **React Router** (Navigation)
- **Vanilla CSS** (Custom responsive styling, clean minimal UI)
- **Chart.js & React-Chartjs-2** (Data Visualization & Reports)
- **Axios** (API Client)

## Features

**Dashboard & Reports**
- Visual analytics and charts displaying task completion and project progress.
- Status filters and quick overviews of pending work.

**Task Management**
- View, create, edit, and update tasks.
- Advanced forms supporting multi-select owners, tags, and dynamic due dates.
- Detailed task view displaying status, tags, and assigned owners.

**Project & Team Organization**
- Group tasks by overarching projects or specific departments/teams.
- Dedicated Team and Project views to intuitively monitor workloads.

**Authentication**
- Secure JWT-based login and registration flows.
